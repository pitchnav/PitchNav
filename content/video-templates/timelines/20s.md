# 20-second timeline

Use for: the most common organic Reel length — enough room for a real
curiosity loop without losing retention.

See `_pacing-techniques.md` for the technique menu referenced below.

| Timecode | Beat | Must be true on screen | Suggested technique |
|---|---|---|---|
| 0:00–0:02 | Pattern-interrupt hook | Strongest visual + hook caption | Hard cut on action |
| 0:02–0:04 | Visual proof | Real pitch footage delivering on the hook | Slow motion in |
| 0:04–0:08 | Problem / observation | The surprising thing, shown at full speed then slow motion | Speed ramp, freeze frame |
| 0:08–0:12 | Curiosity loop / repeated pattern | A second rep or angle that confirms it isn't a one-off | Split screen or angle overlay |
| 0:12–0:16 | Insight or solution | The coaching cue or drill, stated plainly | Diagram insert, caption |
| 0:16–0:18 | Product output | Dashboard/report, unmistakably Pitch Nav | UI screen recording |
| 0:18–0:20 | CTA | "Start your Pitch Nav analysis." | Static end card |

**Notes:** This is the default length for Format A (mechanics breakdown) and
Format F (myth correction). If a drill is shown, keep it to one rep, folded
into the insight beat — don't add a dedicated drill beat at this length.
