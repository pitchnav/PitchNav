# 45-second timeline

Use for: athlete case studies, product walkthroughs, and founder videos where
one product output beat isn't enough room to earn trust.

See `_pacing-techniques.md` for the technique menu referenced below.

| Timecode | Beat | Must be true on screen | Suggested technique |
|---|---|---|---|
| 0:00–0:02 | Pattern-interrupt hook | Strongest visual + hook caption | Hard cut on action |
| 0:02–0:06 | Immediate visual proof | Real pitch footage delivering on the hook | Slow motion in |
| 0:06–0:12 | Reveal the problem or surprising observation | The mechanical detail, shown from a second angle if available | Freeze frame + angle overlay |
| 0:12–0:20 | Curiosity loop / repeated pattern | Second rep or a before/after pairing that confirms the pattern | Split screen |
| 0:20–0:30 | Coaching insight or solution | The idea, plus a drill demonstration (real footage, not described) | Diagram insert, drill B-roll |
| 0:30–0:40 | Product output walkthrough | Upload → phases → priority → drills, moving screen recording, not a static frame | UI screen recording, progress bar |
| 0:40–0:45 | CTA | "Start your Pitch Nav analysis." | Static end card |

**Notes:** This length can carry a named athlete case study (Format B) or a
full founder explanation (Format E) without feeling rushed. Still no dead
air — every beat above should feel like the 30s timeline with one extra beat
inserted, not a slower version of it.
