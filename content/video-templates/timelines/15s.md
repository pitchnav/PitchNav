# 15-second timeline

Use for: fast organic Reels, Collab Reel openers, short paid-ad variants.

See `_pacing-techniques.md` for the technique menu referenced below.

| Timecode | Beat | Must be true on screen | Suggested technique |
|---|---|---|---|
| 0:00–0:02 | Pattern-interrupt hook | Strongest visual + hook caption | Hard cut on action |
| 0:02–0:04 | Visual proof | Real pitch footage delivering on the hook | Slow motion in |
| 0:04–0:07 | Problem / observation | The surprising thing, clearly shown, not just claimed | Freeze frame + joint-marker or angle overlay |
| 0:07–0:10 | Insight or solution | The coaching cue or "why it matters," in one breath | Caption + diagram insert |
| 0:10–0:13 | Product output | Dashboard/report, unmistakably Pitch Nav | UI screen recording |
| 0:13–0:15 | CTA | "Start your Pitch Nav analysis." | Static end card |

**Notes:** No room for a curiosity-loop callback (second rep) at this length
unless it replaces the standalone insight beat — if you use a second rep, fold
it into the problem beat as a quick split screen instead of adding time.
