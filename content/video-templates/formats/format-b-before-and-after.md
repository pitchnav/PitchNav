# Format B — Before and after

**Best for:** athlete case-study videos, progress-tracking content, organic Reels that prove the product works over time.
**Default length:** 30s (see `content/video-templates/timelines/30s.md`); 45s if two movements are being compared.

## Structure

1. Side-by-side opening — both clips visible at once (split screen), so the viewer instantly knows this is a comparison
2. Clear timeframe — on-screen text stating the real gap, e.g. "4 WEEKS APART" (never approximate or invent a timeframe)
3. One movement to watch — tell the viewer exactly where to look before showing the comparison in motion
4. Synchronized comparison — both reps playing at the same speed, ideally synced to the same phase (e.g. both starting at leg lift)
5. What changed — one sentence, specific, tied to the movement named in step 3
6. What still needs work — one honest sentence; this is what makes the case study credible instead of promotional
7. CTA — "Start your Pitch Nav analysis."

## Rules specific to this format

- Only use real footage of a real, consenting athlete. Never simulate a before/after with the same clip slowed down or with an AI-generated "after."
- The timeframe on screen must be accurate. If unsure of the exact gap, use the athlete's actual upload dates.
- "What still needs work" is not optional — a before/after with zero remaining priorities reads as fabricated and undermines trust.
- No fabricated velocity numbers. If velocity is shown, it must be an approved, calibration-backed estimate per `CLAUDE.md` safety rules, or omitted.

## Footage needed

- Two real side-view clips of the same athlete, same camera setup, different dates
- Athlete consent to use their footage and (if applicable) first name
- Optional: a short dashboard clip showing the priority that changed between the two dates

## Caption rhythm example

```
SAME PITCHER
FOUR WEEKS APART
WATCH THE FRONT SIDE
MORE STABLE AT FOOT STRIKE
STILL WORKING ON TIMING
START YOUR ANALYSIS
```
