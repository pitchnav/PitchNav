# Master shoot-day checklist

One page, phone-friendly. Use this to plan and run the day; use the three
detailed checklists for the actual capture list.

## Before the day

- [ ] Objective and script format chosen (`content/production-workflow.md`,
      steps 1-2)
- [ ] Hook(s) selected (`content/video-templates/hook-library.json`)
- [ ] Campaign JSON created from `_TEMPLATE.campaign.json`
- [ ] Athlete/founder availability confirmed, consent forms ready
- [ ] Location scouted for 15-feet side-view camera placement, good lighting
- [ ] Equipment checked: tripod, phone charged, storage space free, backup
      battery

## During the day

- [ ] Camera setup verified against `shot-lists/pitching-footage-checklist.md`
- [ ] Pitching footage captured (full checklist)
- [ ] Founder footage captured, if applicable (full checklist)
- [ ] Screen recordings captured, if not already on hand (full checklist)
- [ ] Consent confirmed and documented

## Before you leave

- [ ] Spot-check playback of at least one clip per setup, in slow motion, on
      a laptop screen
- [ ] Footage backed up to two locations
- [ ] Anything missing flagged now, not discovered in the edit bay

## After the day

- [ ] Campaign JSON `status` updated to `"footage-needed"` → `"shot"`
- [ ] Asset paths filled in under `assetReferences` in the campaign JSON
- [ ] Handed off to editor with the campaign JSON + companion script file
